The Cynthia Ugwu Replica project is a modern, visually engaging, and highly interactive website built using HTML, CSS, JavaScript, and GSAP (GreenSock Animation Platform). It is inspired by the portfolio website of the renowned designer Cynthia Ugwu, known for her sleek minimalism and elegant motion design.

This project focuses on recreating the smooth animations, creative transitions, and responsive design of her site while maintaining performance and clean code practices. Every animation and layout detail has been carefully implemented to reflect a professional-level understanding of front-end development and animation logic.

ğŸ’¡ Project Purpose

The goal of this project was to challenge myself to replicate a high-end creative portfolio website without using frameworks or prebuilt animation libraries other than GSAP. I wanted to explore how animation and interaction can be used to tell a visual story, enhancing user engagement through motion and design.

By rebuilding this site from scratch, I strengthened my knowledge of DOM manipulation, scroll-based animations, and smooth transition effects, while learning how to manage animation performance for different screen sizes.
